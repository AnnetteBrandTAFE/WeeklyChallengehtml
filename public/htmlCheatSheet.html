Cheatsheet

Repository for all wisdom

text tags

<h1></h1> largest headline
<h6></h6> smallest headline
<b></b> defines bold text
<strong></strong> defines test with strong importance
<i></i> is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc
<em></em> defines emphasized text, the content inside is typically displayed in italic
<small></small> defines smaller text
<mark></mark> defines emphasized text, the content inside is typically displayed in italic
<em></em> defines text that should be marked or highlighted
<del></del> defines text that has been deleted from a document
<ins></ins> defines a text that has been inserted into a document

style attributes

<h1 style = "background-color: black;"></h1> defines the background color
<h1 style = "color: red;"></h1> defines the text color
<h1 style = "font-family:verdana;"></h1> defines the font
<h1 style = "font-size: 300%;"></h1> defines the text size
<h1 style = "text-align:center;"></h1> defines the horizontal text alignment

section tags

<p></p> defines a paragraph
<he></he> defines a thematic break in an HTML page, and is most often displayed as a horizontal rule
<br> defines a line break, no closing tag. Use if you want a new line without starting a new paragraph
<pre></pre> defines preformatted text can be used instead of <br>
<blockquote></blockquote> defines a section that is quoted from another source
<q></q> defines a short quotation
<abbr></abbr> defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". Tip: Use the global title attribute to show the description for the abbreviation/acronym when you mouse over the element.
<address></address> defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.).
<cite></cite> defines the contact information for the author/owner of a document or an article

link tags

<a href="URL">link text</a> the <a> defines a hyperlink
<a href="URL" target ="_blank"a>Visit URL!</a> Use target="_blank" to open the linked document in a new browser window or tab
<p><a href="index.html"<a></p> creates a local link (a link to a page within the same website)
<a href="URL"><img src="smiley.png" alt="URL description" style="width: 200px; height:200pc;"></a> to use an image as a link
<a href="mailto:someone@example.com">Send email</a> use mailto: inside the href attribute to create a link that opens the user's email program (to let them send a new email)
<button onclick="document.location='URL'">URL description</button> creates a button you can push
<a href="URL" title="Go to URL???">link text</a> the title attribute specifies extra information about an element. The information is most often shown as a tooltip text when the mouse moves over the element.
<style>a:link{ color: black; background-color: transparent;}</style> to set the link color
<style>a:visited{ color: black; background-color: transparent;}</style> to set the visited link color
<style>a:hover{ color: black; background-color: transparent;}</style> to set the hover link color
<style>a:active{ color: black; background-color: transparent;}</style> to set the active link color
